### Project Title

### Short Description
A full-stack telemedicine web application featuring HD video consultations, real-time chat, and role-based user management. Built with Next.js, Socket.IO, and WebRTC.

### Technologies Used
- **Frontend**: Next.js 16, TypeScript, Tailwind CSS 4
- **Backend**: Node.js, Express, Socket.IO
- **Video**: WebRTC, PeerJS
- **Deployment**: Netlify (Frontend), Render (Backend)

- ‚úÖ HD Video Consultations (1280x720)
- ‚úÖ Real-time Chat System
- ‚úÖ Role-based Access (Doctor/Patient)
- ‚úÖ Professional Zoom-like Interface
- ‚úÖ Responsive Design (Mobile-first)
- ‚úÖ Secure WebRTC Connections
- ‚úÖ Modern UI/UX with Dark Theme

### My Role
**Full-Stack Developer** - Designed and developed the entire application from scratch, including:
- Frontend architecture with Next.js App Router
- Real-time communication with Socket.IO
- WebRTC video integration with PeerJS
- Responsive UI/UX design with Tailwind CSS
- Deployment and DevOps setup

### Challenges Solved
1. **WebRTC Connectivity** - Implemented STUN servers for NAT traversal
2. **Real-time Sync** - Built bidirectional communication with Socket.IO
3. **State Management** - Managed complex video call states with React hooks
4. **Responsive Design** - Created mobile-first, adaptive layouts
5. **Production Deployment** - Configured separate frontend/backend hosting

### Impact
- Enables remote healthcare consultations
- Reduces patient travel time and costs
- Provides secure, HIPAA-ready platform
- Improves healthcare accessibility
